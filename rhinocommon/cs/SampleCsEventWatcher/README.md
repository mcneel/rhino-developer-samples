SampleCsEventWatcher
====================

Sample C# RhinoCommon plug-in that demonstrates how to use event watchers.

Building Sample
--------------------
To build the sample, you are going to need:

* Rhinoceros 8 (http://www.rhino3d.com)
* Microsoft Visual C# 2022
